# Using a React frontend with Vaadin TypeScript Endpoints

This is a demo of using React views to connect to a Vaadin TS backend.

## Project structure

- `/frontend` - all frontend sources
- `/src` - all backend sources

## Running the app

`mvn spring-boot:run`
